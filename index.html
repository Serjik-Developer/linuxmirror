Quick Configuration

Edit /etc/apt/sources.list (или /etc/apt/sources.list.d/linuxmirror.list):

# Ubuntu 24.04 (Noble) — /etc/apt/sources.list
deb https://linuxmirror.host/ubuntu noble main restricted universe multiverse
deb https://linuxmirror.host/ubuntu noble-updates main restricted universe multiverse
deb https://linuxmirror.host/ubuntu-security noble-security main restricted universe multiverse

# arm64/armhf — ports
deb https://linuxmirror.host/ubuntu-ports noble main restricted universe multiverse

Edit /etc/pacman.d/mirrorlist:

## linuxmirror.host
Server = https://linuxmirror.host/archlinux/$repo/os/$arch

Create /etc/yum.repos.d/linuxmirror.repo:

[linuxmirror-fedora]
name=linuxmirror.host - Fedora $releasever - $basearch
baseurl=https://linuxmirror.host/fedora/releases/$releasever/Everything/$basearch/os/
enabled=1
gpgcheck=1

Edit /etc/apk/repositories:

https://linuxmirror.host/alpine/v3.20/main
https://linuxmirror.host/alpine/v3.20/community
